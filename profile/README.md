# 모구픽(MOGUPICK)
- **모** : ‘모아’ — 파편화된 구독 상품을 한 곳에 모으다
- **구** : ‘구독’ — 다양한 카테고리의 구독 상품
- **픽** : ‘Pick’ — 나에게 맞는 구독을 고르고 선택하다
